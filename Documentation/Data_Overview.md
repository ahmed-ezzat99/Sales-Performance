# Data Overview

This project is based on a financial dataset sourced from an Excel file.

## Dataset Description

The dataset contains financial performance data across multiple dimensions, including:
- Country
- Segment
- Product
- Sales
- Profit
- Date (Year / Month)

## Data Source

- Excel spreadsheet containing historical sales and profit data.

## Data Preparation

- Data cleaning and transformation were performed using Power Query.
- Missing values and inconsistent formats were handled.
- Columns were standardized to ensure accurate analysis.
- The dataset was structured to support proper relationships and DAX calculations.
