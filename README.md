# Power BI Project  
## Sales Performance  

## Project Objective  
The objective of this project is to analyze sales and profit performance across countries, segments, and products using Power BI.  
The focus is on identifying revenue drivers, profit leaders, and understanding the gap between total sales and profitability.

This project demonstrates key skills relevant to Data Analyst and BI Analyst roles, including data cleaning, DAX calculations, and dashboard development.

## Project Scope  
- Analyze total sales and profit by country  
- Compare sales volume versus profitability  
- Evaluate profit performance by segment  
- Identify top and low-margin products  
- Track sales and profit trends over time  

## Tools & Technologies  
- Power BI Desktop  
- Power Query  
- DAX for measures and KPIs  
- Data modeling and relationships  

## Key KPIs  
- Total Sales  
- Total Profit  
- Profit Margin %  
- Sales by Country  
- Profit by Segment  

## Project Deliverables  
- Cleaned and modeled dataset in Power BI  
- Interactive financial dashboard  
- Business-focused insights and findings  

## Key Insights  
- The United States leads in total sales volume  
- France achieved the highest profit despite lower sales  
- The Government segment is the most profitable  
- A clear gap exists between total sales (118.7M) and profit (16.89M)
